# @refraction-ui/angular-data-table

## 0.1.0

### Minor Changes

- Initial release of `angular-data-table` component.
  - `DataTableComponent` (`re-data-table`) wrapping the headless `@refraction-ui/data-table` core.
  - Supports sorting (click column headers), per-column filtering (inline inputs), and reactive state via Angular signals.
  - Outputs: `sortChange`, `filterChange`.
