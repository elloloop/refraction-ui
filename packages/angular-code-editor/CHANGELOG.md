# @refraction-ui/angular-code-editor

## 0.1.0

### Minor Changes

- Initial release of `angular-code-editor` component.
  - `CodeEditorComponent` (`re-code-editor`) wrapping the headless `@refraction-ui/code-editor` core.
  - Supports `value`, `language`, `readOnly`, and `theme` inputs.
  - Basic textarea-based rendering with accessibility support.
