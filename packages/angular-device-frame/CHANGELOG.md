# @refraction-ui/angular-device-frame

## 0.1.0

### Minor Changes

- Initial release of `angular-device-frame` component.
  - `DeviceFrameComponent` (`re-device-frame`) wrapping the headless `@refraction-ui/device-frame` core.
  - Supports iPhone, iPad, and Android frames.
  - Supports portrait and landscape orientations.
  - Includes simulated notch and home indicator for supported devices.
