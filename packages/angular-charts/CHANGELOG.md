# @refraction-ui/angular-charts

## 0.1.0

### Minor Changes

- Initial release of `angular-charts` package.
  - Implements declarative data visualization components wrapping the headless `@refraction-ui/charts` core.
  - Includes `<re-chart>`, `[re-bars]`, `[re-line]`, `[re-circles]`, `[re-x-axis]`, `[re-y-axis]`, `[re-gradient]`, `[re-histogram]`, `[re-scatter-plot]`, `[re-pie-chart]`.
