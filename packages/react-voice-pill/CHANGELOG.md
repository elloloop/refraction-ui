# @refraction-ui/react-voice-pill

## 0.2.0

### Minor Changes

- Add voice-focused primitives for AI and audio interfaces.

  - `@refraction-ui/react-voice-pill` adds a fixed-position speaker status pill with safe-area-aware placement, intensity-driven pulse rings, speaker keyed theming, polite live-region metadata, and an optional icon mute toggle.
  - `@refraction-ui/react-waveform` adds a canvas waveform visualizer for analyser nodes, media streams, sample arrays, and low-cost intensity streams with resize-aware rendering, smoothing, paused state, reduced-motion behavior, and bars / line / rings variants.

### Patch Changes

- Updated dependencies
  - @refraction-ui/voice-pill@0.2.0
