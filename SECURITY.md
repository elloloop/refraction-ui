# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| canary  | :white_check_mark: |

## Reporting a Vulnerability

We take security issues seriously. If you discover a security vulnerability, please report it responsibly.

**Please do NOT open a public issue for security vulnerabilities.**

Instead, please use [GitHub Security Advisories](https://github.com/elloloop/refraction-ui/security/advisories/new) to report vulnerabilities privately.

### What to expect

- **Acknowledgment**: We will acknowledge your report within 48 hours.
- **Updates**: We will provide updates on the progress of the fix.
- **Disclosure**: We will coordinate with you on the timing of public disclosure.

### What to include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)
